# PROJETO DIGIMON

## projeto desenvolvido com framework react + typescript

### clonar projeto do git

```
git clone <url do repositorio>
```

### instalar dependecias

```
npm install
```

ou

```
yarn
```

### rodar projeto

```
yarn dev
```

ou

```
npm run dev
```
